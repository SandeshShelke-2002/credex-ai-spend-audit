# AI Spend Audit

AI Spend Audit is a free tool for startup founders and engineering teams to analyze AI subscription costs and identify savings opportunities.

## Features
- AI tool spend audit
- Savings recommendations
- Annual/monthly savings
- Lead capture with Supabase
- Persistent form state

## Tech Stack
- Next.js
- Tailwind CSS
- Supabase
- Vercel

## Run locally
## bash or terminal
- npm install
- npm run dev
